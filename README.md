
# VWAP Calculator in C

**VWAP Calculator** is a C-based program that computes the **Volume Weighted Average Price (VWAP)** from historical stock market data. VWAP is widely used by traders and analysts to evaluate the average price of a stock weighted by its trading volume throughout the day. This project focuses on accurate calculation and analysis using standard C programming.

## Features
- Reads stock data from a CSV file containing **Date, Open, High, Low, Close, and Volume**.
- Calculates daily VWAP using standard financial formulas.
- Provides insights into trading trends and price movements.
- Lightweight and modular C code for easy understanding and modification.

## Technologies Used
- **C Programming**
- File handling for CSV input/output
- Standard financial computations
